List of HTML Tags:

  1. h1: It defines the heading with level 1
  2. p: It defines the paragraph
  3. div: It defines a container
  4. br: It is used to break the line
  5. b: It is used to bold any content
  6. i: It is used to do any content italic
  7. table: It defines table
  8. tr: It defines table row
  9. td: It defines table data
  10. th: It defines table heading
  11. img: It defines an image
